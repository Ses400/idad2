Technical response

The game is implemented as a grid of 25 interactive buttons (colored squares) where each button is represented through a grid and assigned an ID so it can be individually controlled by JavaScript. The overall system ties together random sequence generation, user input, and audio-visual feedback. The advanced technique of Math.random() is used in the generateNewSequence() function to select random button sequences for each round. Every new level extends the sequence by one random value, ensuring the game becomes progressively more challenging. Continuing on, for the user input, the player clicks one of the grid squares and the index of the button is added to the array. Then each button flashes to identify which sequence the user needs to memorise and it triggers a sound via the playSound() function). Inputs are compared against the generated sequence and if the input matches, the game continues and if the input fails, the whole grid flashes red and a mistake sound plays. For the visual feedback there are flashing squares (white for every system sequence and red for mistakes). There are also different sounds for correct completions vs. mistakes and toast notifications display short updates like success or failure.
