Conceptual Response

My chosen creative prompt for the project was “no text characters” where I only used shapes and sounds to relay a message, and my advanced technique was Math.random() where I randomised pattern sequences. I designed a pattern recognition memory game inspired by Simon says, where the system plays a random sequence of flashing squares and the player’s task is to repeat the pattern by clicking the same buttons in the correct order. Each round adds a new random element to the sequence, making the challenge progressively harder. By using only colors, flashing effects, and sounds, the game avoids text based cues and relies entirely on visual and auditory memory. This created a minimalistic and shape driven interaction that aligns with the “no text characters” constraint and focuses on intuitive recognition and recall.
